KEEP-IT-SIMPLE

Password Manager

SYSTEM INFORMATION

DATE CREATED: FEBRUARY 2021

PROGRAMMING LANGUAGE USED: PHP, MySQL, JS

KEEP-IT-SIMPLE is a Password Manager website developed as the final requirement for our major subject. This password manager is designed to securely store the login credentials of various user accounts. It is especially useful for individuals who have numerous accounts and need an efficient way to manage and remember their passwords. KEEP-IT-SIMPLE helps users maintain better security practices and simplifies the process of accessing their online accounts.
